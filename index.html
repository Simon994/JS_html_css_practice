Forms

<Form> element is used to collect and send info to an external source. Forms can contain various input elements. The form's action attribute is a URL, speicifying where the info from the input(s) should be passed. The form's method attribute (assigned an HTTP verb; e.g. 'PUT', 'POST', 'GET') specifies what the form should do with the info once it is sent
